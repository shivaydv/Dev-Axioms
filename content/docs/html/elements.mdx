# HTML Elements & Tags

Understanding HTML elements and their proper usage is crucial for building well-structured web pages.

## Block vs Inline Elements

<Callout type="info">
Block elements start on a new line and take full width, while inline elements flow within the content.
</Callout>

### Common Block Elements
- `<div>`: Generic container
- `<p>`: Paragraph
- `<h1>` to `<h6>`: Headings
- `<section>`: Thematic grouping
- `<article>`: Independent content

### Common Inline Elements
- `<span>`: Generic inline container
- `<a>`: Hyperlink
- `<strong>`: Strong importance
- `<em>`: Emphasized text
- `<img>`: Image

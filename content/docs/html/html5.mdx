# HTML5 Features

HTML5 introduced many powerful features and APIs that revolutionized web development.

## New Semantic Elements

<Callout type="info">
HTML5 introduced semantic elements to better structure web content:
- `<header>`, `<footer>`, `<nav>`
- `<main>`, `<article>`, `<section>`
- `<figure>`, `<figcaption>`
- `<time>`, `<mark>`, `<details>`
</Callout>

## Common Interview Questions

### 1. What is the difference between HTML and XHTML?

<Callout>
HTML is more forgiving in terms of syntax, while XHTML follows strict XML rules:
- XHTML elements must be properly nested
- XHTML elements must always be closed
- XHTML elements must be in lowercase
- XHTML documents must have one root element
</Callout>

### 2. Explain the importance of DOCTYPE

The DOCTYPE declaration:
- Tells the browser which version of HTML the page uses
- Ensures consistent rendering across browsers
- Triggers standards mode vs quirks mode

## Common Interview Questions

### 1. Explain Event Loop in JavaScript

<Callout>
The Event Loop is JavaScript's mechanism for handling asynchronous operations:
- Call Stack: Executes synchronous code
- Callback Queue: Holds callback functions
- Microtask Queue: Handles Promises
- Event Loop: Monitors stack and queues
</Callout>
